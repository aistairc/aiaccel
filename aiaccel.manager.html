all_parameters_processed(num_ready: int, num_running: int) → bool[ソース]

Checks whether any unprocessed parameters are left.

This method is beneficial for the case that the maximum number of parameter generation is limited by algorithm (e.g. grid search). To make this method effective, the algorithm with the parameter generation limit should turn all_parameters_generated True when all of available parameters are generated.

パラメータ:

  • num_ready (int) -- The number of ready jobs.

  • num_running (int) -- The number of running jobs.

戻り値:

True if all parameters are generated and are processed.

bool
